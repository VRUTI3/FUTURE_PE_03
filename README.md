## 📈 Task 3 – AI SEO Blog & Content Cluster Generator

This task demonstrates how prompt engineering can generate SEO-focused content strategies for business websites.

The system generates:

• Pillar blog articles  
• Supporting blog topics  
• SEO blog structures (H1–H3)  
• Internal linking strategies  
• Local SEO optimization  

This approach follows the content cluster model used by SEO agencies and SaaS companies.
